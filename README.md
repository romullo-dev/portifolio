# Portfólio - Rômulo França

Portfólio pessoal desenvolvido para apresentar perfil profissional, tecnologias, projetos de programação e canais de contato. O foco do site é servir como uma vitrine simples, moderna e responsiva para recrutadores e oportunidades na área de desenvolvimento.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub
- GitHub Pages

## Projetos apresentados

### DNATransportes

Projeto principal do portfólio. É apresentado como um case profissional de gestão logística, com foco em:

- Pedidos
- Rotas
- Status e rastreabilidade
- Documentos
- Dashboard
- Apoio mobile

### Financias

Financias — plataforma de controle financeiro inteligente com dashboard, receitas, despesas, reservas, gastos fixos, notificações, autenticação e segurança por usuário usando Supabase e PostgreSQL.

### Sistema de Controle de Estoque

Projeto em desenvolvimento para controle operacional de produtos, entradas, saídas, inventário e movimentações internas.

## Como executar

Este projeto não precisa de instalação ou build. Basta abrir o arquivo `index.html` no navegador.

Para testar com um servidor local, use qualquer servidor estático simples apontando para a raiz do projeto.

## Estrutura principal

- `index.html`: página inicial do portfólio.
- `CSS/style.css`: estilos globais do site.
- `assets/js/main.js`: interações simples, como menu mobile e animações de entrada.
- `assets/img/`: imagens do perfil e dos projetos.
- `projects/dna-transportes.html`: página de case do DNATransportes.
- `projects/financias.html`: página de case do Financias.
- `projects/estoque.html`: página do projeto de controle de estoque.

## GitHub Pages

O projeto está pronto para GitHub Pages por ser totalmente estático. Para publicar:

1. Envie o projeto para um repositório no GitHub.
2. Acesse as configurações do repositório.
3. Em Pages, selecione a branch principal e a pasta raiz.
4. Salve e aguarde o link de publicação.

## Ajustes recomendados

- Atualizar links de repositórios específicos quando os projetos estiverem públicos.
- Adicionar link de WhatsApp, se quiser usar contato direto.
- Atualizar prints dos projetos conforme novas telas forem finalizadas.
- Adicionar currículo em PDF, caso queira incluir um botão de download no futuro.
